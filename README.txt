Рекомендованные аргументы запуска:

./lab11avkN32471.elf . 0x596F7557696C6C4E6576657246696E644D65
(проверка поиска по текстовым файлам - находит все экземпляры строки YouWillNeverFindMe)

(проверка поиска по изображениям. Каждый маркер секции .png начинается с 0xFF)
